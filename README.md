#Password-manager

1. git clone git@github.com:Torenciel/Password-Manager.git
2. entrer : npm i

3. Dans le fichier .env ajouter ;

DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
PORT=

JWT_SECRET=
JWT_EXPIRES_IN=

BREVO_SMTP_HOST=
BREVO_SMTP_PORT=
BREVO_SMTP_USER=
BREVO_SMTP_PASS=

4. Variables CRUD

- Authentification:
registerUser
verifyUserEmail
login
resetPasswordRequest
resetPassword

- Password manager
createPassword
getAllPasswords
getPasswordById
updatePassword
deletePassword
